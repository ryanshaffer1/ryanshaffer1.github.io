Fantasy Projections

A Neural Network that accurately predicts an NFL player’s final game stats and Fantasy Football score, and updates that prediction as the game progresses.
Not yet beating Vegas odds, but maybe good enough to beat your coworkers in the office Fantasy league.

Project Description

This project was my first foray into Machine Learning. I started by listing out the variables that a human might use to predict ahead of time whether an NFL player will do well or poorly: the player's name, their team and opponent, the record of both teams, their stats from previous games, etc. I added the variables that would influence this prediction as the game plays out: the game score, who has the ball, the player's stats so far... Before long, there were nearly 400 unique variables taken into account.

These variables can be taken from online sources for every past NFL game in the last several decades. I've only looked at the 2021-2023 period, and focused on the 300 top NFL players by Fantasy Football score. That still leaves me with 610,000 unique mid-game Fantasy score predictions to make. Since we also know the true Fantasy score for every player in past games, we can use the true data to train a Neural Net to identify patterns in which players tend to excel in which types of stats (passing, rushing, etc). The resulting predictions can be surprisingly good - check out the accuracy metrics below.

The Neural Net is fairly rudimentary, with the baseline configuration holding just under 50k learnable parameters. I've found an initial embedding layer that keeps categorical data (e.g. Player ID, Team ID) separated to be effective in extracting some meaning out of those categories (i.e., which players are "good" at certain stats). The biggest contributor to Neural Net effectiveness (measured by average Fantasy point difference between prediction and truth) so far has been optimizing the Net's hyper-parameters using a recursive grid search.

The source code is written entirely in Python, with a strong focus on generalization to new ML and optimization techniques, new configurations (like different Fantasy Football rules), and eventual application to a user-friendly Fantasy Football manager and simulator app.

Through this project, I've learned the theory and basic implementation of Neural Networks, a core component of the ML/AI capabilities that are now everywhere. I've gotten much better at writing Python and at general software development. I'm working on robust testing, documentation, continuous integration... just don't call me a Software Engineer yet!
